# internal-ip-cli

> Get your internal IP address


## Install

```
$ npm install --global internal-ip-cli
```


## Usage

```
$ internal-ip --help

  Usage
    $ internal-ip

  Options
    --ipv6 -6  Return the IPv6 address (Default)
    --ipv4 -4  Return the IPv4 address

  Examples
    $ internal-ip
    fe80::1
    $ internal-ip --ipv4
    10.0.0.79
```
